"ArbiNexus detects when on-chain market price diverges from oracle fair value on Solana. Most users miss these windows because data is fragmented and timing is tight."

- Open dashboard
- Point to health badges for Pyth and Jupiter
- Call out that both providers are live
- Point to trust panel and decision rules

Line to say:
"This panel is here so users and judges can verify data quality before trusting any signal."

## 0:40 to 1:15 - Opportunity Analysis

- Highlight top row in opportunities table
- Explain spread, net, signal, and rationale columns
- Toggle actionable-only filter

Line to say:
"We do not show everything as actionable. We classify and filter by net edge and confidence."

## 1:15 to 1:40 - Simulation

- Click Simulate Trade on an actionable row if available
- Show inline result: entry, target, net, estimated return, route

If result is negative, say:
"This pair is efficiently priced right now, so net is negative. That is expected and prevents false positives."

## 1:40 to 2:00 - Vision

"Today this is a paper-trade-first intelligence layer. Next it evolves toward safer automated execution and broader tokenized asset coverage on Solana."
