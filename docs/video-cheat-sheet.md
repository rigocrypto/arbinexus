# ArbiNexus Video Cheat Sheet

## 5-Line Short Narration
1. ArbiNexus is oracle-informed arbitrage intelligence for Solana.
2. We compare Pyth fair value against live executable Jupiter quotes.
3. We compute net edge after fees and classify actionable opportunities.
4. Every signal includes rationale and confidence context for transparency.
5. Before any execution, we run paper-trade simulation for safe decision support.

## 2-Minute Script Bullets
- 0:00 to 0:10: Intro
  - "This is ArbiNexus, an explainable arbitrage intelligence dashboard for Solana."
- 0:10 to 0:25: Problem
  - "Market prices can diverge from oracle fair value briefly. Most users cannot detect and validate those windows quickly, and many bots are opaque."
- 0:25 to 0:45: Core solution
  - "ArbiNexus streams Pyth oracle prices and Jupiter executable quotes, computes spread and net edge after fees, then classifies each signal into BUY_MARKET, SELL_MARKET, or NO_TRADE."
- 0:45 to 1:10: Live dashboard walkthrough
  - Show health badges and live status.
  - Show opportunities table columns: spread, net, signal, confidence, rationale.
  - Mention explainability panel and trust indicators.
- 1:10 to 1:35: Simulation flow
  - "Now I simulate one opportunity. This gives a paper-trade outcome before any capital is committed."
  - Run simulate and read key output.
- 1:35 to 1:50: Why Solana
  - "Solana gives us low-latency market access, deep routing via Jupiter, and robust oracle infrastructure via Pyth."
- 1:50 to 2:00: Close
  - "ArbiNexus makes arbitrage opportunities understandable, transparent, and safer to evaluate in real time."

## Fallback Line for Negative Simulation Result
- "This is a useful outcome: simulation prevented a bad trade, which is exactly why ArbiNexus emphasizes explainability and paper-trade validation before execution."

## Recording Tips (Fast)
- Keep browser zoom around 90% to 100% so key cards and table rows fit on screen.
- Record at 1080p if possible.
- Keep one preselected row ready for simulation to avoid dead time.
- If live market data is quiet, focus narration on explainability and risk filtering.
- End on the trust panel and one clear summary sentence.
